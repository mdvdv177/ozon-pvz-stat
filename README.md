## Быстрые отчёты по отправлениям на складе ПВЗ Ozon
**Настройка:**
- должен быть установлен Python, библиотеки pandas, openpyxl (`pip install pandas openpyxl`)
- скачиваете `script.py`, в пустую папку (там будет появляться csv-файл БД склада)
- для MacOS настоятельно рекомендую скачать файл автоматизации (Quick Action), чтобы использовать в два клика из Finder (нужно будет открыть файл в Automator и указать путь к `script.py`, там же желательно изменить `python3` на полный путь до интерпретатора)

**Использование:**
- скачиваем отчет в формате *.xslx* (Турбо ПВЗ -> Отчёты -> Остатки на складе)
- `python(или python3) script.py <путь к xlsx файлу>`
- для MacOS, если создан Quick Action, то просто клик ПКМ по файлу -> Quick Actions -> process storage
- программа выведет HTML в формате ***<поряд. № заказа> (всего товаров): яч n - x, яч m - y***
